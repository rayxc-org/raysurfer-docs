---
title: Store Code Block
api: POST /v1/store
description: "Store a code block for future retrieval"
---

## Request

<ParamField body="source" type="string" required>
  The source code content to store
</ParamField>

<ParamField body="entrypoint" type="string" required>
  The main function or entry point name
</ParamField>

<ParamField body="language" type="string" required>
  Programming language (python, typescript, javascript, etc.)
</ParamField>

<ParamField body="task_description" type="string" required>
  Natural language description of what this code does
</ParamField>

<ParamField body="metadata" type="object">
  Optional metadata for the code block
</ParamField>

<ParamField body="private" type="boolean" default="false">
  Keep this code block private to your organization (Pro/Enterprise only)
</ParamField>

## Example Request

```bash
curl -X POST https://api.raysurfer.com/v1/store \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "source": "def generate_invoice(order):\n    items = [format_line_item(i) for i in order.items]\n    html = render_template(\"invoice.html\", order_id=order.id, items=items)\n    return html_to_pdf(html)",
    "entrypoint": "generate_invoice",
    "language": "python",
    "task_description": "Generate a PDF invoice from an order object"
  }'
```

## Response

```json
{
  "success": true,
  "data": {
    "id": "cb_abc123xyz",
    "created_at": "2025-01-18T12:00:00Z",
    "embedding_dimensions": 1024
  },
  "meta": {
    "request_id": "req_def456",
    "latency_ms": 120
  }
}
```

## Response Fields

<ResponseField name="id" type="string">
  Unique identifier for the stored code block
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the block was stored
</ResponseField>

<ResponseField name="embedding_dimensions" type="number">
  Number of dimensions in the generated embedding vector
</ResponseField>
