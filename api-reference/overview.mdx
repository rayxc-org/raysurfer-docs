## Base URL

```
https://api.raysurfer.com/v1
```

## Authentication

All API requests require an API key passed in the `Authorization` header:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.raysurfer.com/v1/retrieve
```

Get your API key from the [dashboard](https://raysurfer.com/dashboard/api-keys).

## Rate Limits

| Plan | Requests/minute | Stored blocks |
|------|-----------------|---------------|
| Free | 60 | 1,000 |
| Pro | 600 | 100,000 |
| Enterprise | Unlimited | Unlimited |

## Response Format

All responses are JSON with this structure:

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "request_id": "req_abc123",
    "latency_ms": 45
  }
}
```

## Error Handling

Errors return appropriate HTTP status codes:

| Code | Description |
|------|-------------|
| 400 | Bad request - invalid parameters |
| 401 | Unauthorized - invalid or missing API key |
| 404 | Not found - no matching code blocks |
| 429 | Rate limit exceeded |
| 500 | Internal server error |

Error response format:

```json
{
  "success": false,
  "error": {
    "code": "RATE_LIMITED",
    "message": "Rate limit exceeded. Try again in 30 seconds."
  }
}
```
