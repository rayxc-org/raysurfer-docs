## Installation

```bash
pip install raysurfer
```

## Setup

Set your API key:

```bash
export RAYSURFER_API_KEY=your_api_key_here
```

Get your key from the [dashboard](https://raysurfer.com/dashboard/api-keys).

## Usage

Just change your import. Everything else stays the same.

```python
# Before
from claude_agent_sdk import ClaudeSDKClient, ClaudeAgentOptions

# After
from raysurfer import ClaudeSDKClient, ClaudeAgentOptions

# Your existing code works unchanged
async with ClaudeSDKClient(options) as client:
    await client.query("Generate quarterly report")
    async for msg in client.receive_response():
        print(msg)
```

## Configuration

Optional settings you can add:

```python
from raysurfer import ClaudeSDKClient, RaysurferOptions

options = RaysurferOptions(
    # Raysurfer options
    cache_enabled=True,      # Turn caching on/off (default: True)
    private_outputs=False,   # Keep outputs private (Pro/Enterprise)

    # Standard Claude SDK options
    model="claude-sonnet-4-20250514",
    max_tokens=4096,
)

async with ClaudeSDKClient(options) as client:
    # ...
```

## Disable Caching for a Request

```python
# Skip cache for this specific query
await client.query("Generate unique content", cache=False)
```

## Check Cache Status

```python
response = await client.query("Generate report")

if response.cache_hit:
    print("Served from cache!")
```
