## Installation

```bash
npm install raysurfer
```

## Setup

Set your API key:

```bash
export RAYSURFER_API_KEY=your_api_key_here
```

Get your key from the [dashboard](https://raysurfer.com/dashboard/api-keys).

## Usage

Just change your import. Everything else stays the same.

```typescript
// Before
import { ClaudeSDKClient } from "@anthropic-ai/claude-agent-sdk";

// After
import { RaysurferClient } from "raysurfer";

// Your existing code works unchanged
const client = new RaysurferClient(options);

for await (const msg of client.query("Generate quarterly report")) {
  console.log(msg);
}
```

## Configuration

Optional settings you can add:

```typescript
import { RaysurferClient, RaysurferOptions } from "raysurfer";

const options: RaysurferOptions = {
  // Raysurfer options
  cacheEnabled: true,      // Turn caching on/off (default: true)
  privateOutputs: false,   // Keep outputs private (Pro/Enterprise)

  // Standard Claude SDK options
  model: "claude-sonnet-4-20250514",
  maxTokens: 4096,
};

const client = new RaysurferClient(options);
```

## Disable Caching for a Request

```typescript
// Skip cache for this specific query
await client.query("Generate unique content", { cache: false });
```

## Check Cache Status

```typescript
const response = await client.query("Generate report");

if (response.cacheHit) {
  console.log("Served from cache!");
}
```
