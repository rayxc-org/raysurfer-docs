Raysurfer enforces two layers of rate limiting: **request quotas** (per plan tier) and **HTTP rate limits** (per endpoint).

## Request Quotas

Each plan tier has a maximum number of API requests per billing period. Once exceeded, the API returns `429 Too Many Requests` until the period resets.

| Tier | Quota | Period | Reset |
|------|-------|--------|-------|
| **Free** | 100 requests | Monthly | 1st of each month |
| **Pro** | 10,000 requests | Daily | Midnight UTC |
| **Enterprise** | 100,000 requests | Daily | Midnight UTC |

### Quota Exceeded Response

When you exceed your quota, the API returns:

```json
{
  "error": "quota_exceeded",
  "message": "You have used 100/100 requests this month. Upgrade your plan for higher limits.",
  "tier": "free",
  "limit": 100,
  "period": "monthly",
  "usage": 100
}
```

## HTTP Rate Limits

Independent of quotas, all tiers share per-minute rate limits on API endpoints. These prevent burst abuse and ensure fair access.

| Endpoint Category | Limit | Endpoints |
|-------------------|-------|-----------|
| **Retrieve** | 200/minute | `/api/retrieve/code-blocks`, `/api/retrieve/best-for-task`, `/api/retrieve/executions`, `/api/retrieve/few-shot-examples`, `/api/retrieve/task-patterns`, `/api/retrieve/code-files` |
| **Store** | 30/minute | `/api/store/code-block`, `/api/store/execution`, `/api/store/execution-result`, `/api/store/auto-review`, `/api/store/cache-usage`, `/api/store/synthetic-queries` |
| **Default** | 100/minute | All other endpoints |

## Plan Limits

Each tier also sets limits on organization resources.

| Tier | Max Members | Max Workspaces | Namespace |
|------|-------------|----------------|-----------|
| **Free** | 1 | 0 | Shared |
| **Pro** | 10 | 0 | Dedicated |
| **Enterprise** | 100 | 50 | Dedicated (per org + per workspace) |

## Handling Rate Limits

When rate limited, implement exponential backoff:

<CodeGroup>

```python Python
import asyncio
from raysurfer import RaySurfer
from raysurfer.exceptions import APIError

client = RaySurfer(api_key="your_api_key")

async def fetch_with_retry(task: str, max_retries: int = 3):
    for attempt in range(max_retries):
        try:
            return client.get_code_snips(task=task)
        except APIError as e:
            if e.status_code == 429 and attempt < max_retries - 1:
                await asyncio.sleep(2 ** attempt)
            else:
                raise
```

```typescript TypeScript
import { RaySurfer, APIError } from "raysurfer";

const client = new RaySurfer({ apiKey: "your_api_key" });

async function fetchWithRetry(task: string, maxRetries = 3) {
  for (let attempt = 0; attempt < maxRetries; attempt++) {
    try {
      return await client.getCodeSnips({ task });
    } catch (e) {
      if (e instanceof APIError && e.statusCode === 429 && attempt < maxRetries - 1) {
        await new Promise((r) => setTimeout(r, 2 ** attempt * 1000));
      } else {
        throw e;
      }
    }
  }
}
```
